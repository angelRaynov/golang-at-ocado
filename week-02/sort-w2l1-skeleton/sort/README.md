# sort-system-v1

This is the first version of the [SORT system](https://www.youtube.com/watch?v=BQDliV7w7_8) we'll be building in the course.

## Steps to do
 * (Optional) Modify the following files and change the repository reference from `github.com/bbsbb/go-at-ocado/sort-vX` to your own repo:
   * `go.mod`
   * `Makefile.GRPC`
   * all files in the `idl` directory 
   * `sorting-service/go.mod`, `sorting-service/main.go`, `sorting-service/service.go`

## How to run projects
 * Enter the respective folder and type `go run *.go`
